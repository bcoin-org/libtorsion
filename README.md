# ec

Experimental EC implementation.

Only utilizes reduction functions generated by [fiat-crypto] (which have proofs
of correctness, formal verification, etc).

## Current Features

- Curve Support:
    - p192
    - p224
    - p256
    - p384
    - p521
    - secp256k1
    - curve25519
    - curve448
    - edwards25519
    - edwards448

- Schemes:
    - ECDSA
    - ECDH
    - EdDSA
